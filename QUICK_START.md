## 🚀 **Start the App**

### Frontend (React)
```bash
cd frontend
npm run dev
```
**URL:** http://localhost:5173

### Backend (Spring Boot) - Optional
```bash
cd backend
run.bat
```
**URL:** http://localhost:8080/api

## 🎯 **How to Use**

1. **Dashboard**: View overview, take medications, see today's reminders
2. **Medications**: Add your medications with dosage and stock info
3. **Reminders**: Set up medication schedules (linked to your medications)
4. **Analytics**: View your adherence trends and progress
5. **AI Insights**: Chat with AI for health guidance

## 🌙 **Theme Toggle**
- Click the moon/sun icon in the navbar
- Theme preference is automatically saved
- All components support both light and dark modes

## 💾 **Data Storage**
- All data is stored locally in your browser
- No account needed - works offline
- Data persists between sessions
